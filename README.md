# Survival 2.0

A simple **Unity-based crafting and survival game** where players can gather resources, craft tools, and explore the environment.  

---

## Features

- 🌳 **Resource Gathering:** Chop down trees to collect logs and rocks.  
- 🪓 **Crafting System:** Combine logs and rocks in your inventory to create tools and weapons.  
- 🧰 **Inventory System:** Keep track of resources and crafted items.  
- 👀 **Look Info Feature:** Hover over or select objects to see their names and details in text.  
- 🏹 **Survival Gameplay:** Use crafted tools to interact with the environment and survive.  

---

## How to Play

1. **Move your character** around the environment.  
2. **Collect resources** like trees and rocks.  
3. **Open your inventory** to craft tools and weapons.  
4. **Use tools** to chop down trees and gather more materials.  
5. **Look at objects** to see information about them in text.  

---

## Development

This game was built using **Unity**.  

### Core Scripts and Systems

– Handles item storage and crafting logic.  
– Combines resources into tools or weapons.  
- Allows objects to display text when looked at.  
– Handles player movement and actions.  

### Assets

- Trees, rocks, and logs for resource gathering.  
- Basic weapons and tools for crafting.  
- UI elements for inventory and look info display.
